# 🎤 Tailored Voice Activation for Enhanced Accessibility

Multilingual AI Voice Assistant designed for elderly and disabled people. Supports voice interaction in Telugu, Hindi, and English.

---

## 🚀 Features

* 🎤 Speech-to-Text conversion
* 🌐 Multilingual support (Telugu, Hindi, English)
* 🤖 AI-based chatbot responses
* 📊 Model comparison and evaluation
* 🧠 NLP-based text processing

---

## 🛠️ Tech Stack

* Python
* NLP (Natural Language Processing)
* Machine Learning
* Speech Recognition
* HTML, CSS (Frontend)

---

## 📂 Project Structure

* `app.py` – Main application
* `speech_to_text.py` – Converts voice to text
* `translate_lahaja.py` – Handles multilingual translation
* `evaluate_transcriptions.py` – Model evaluation
* `templates/` – UI files
